# Desafio 01 — Sistema Bancário Básico

Este primeiro desafio apresenta a versão inicial do sistema bancário em Python.

## Objetivo

Construir uma aplicação simples que permita:

- depositar valores positivos,
- sacar respeitando limite diário e saldo disponível,
- exibir o extrato das movimentações.

## Funcionalidades

- Depósito com validação de valor positivo
- Saque com limite diário de 3 operações
- Saque respeitando o limite máximo de R$ 500,00
- Exibição de extrato com todas as transações

## Como usar

1. Abra o terminal na pasta `desafio-01`
2. Execute:

```bash
python desafio_bancario_basico.py
```

## Estrutura

- `desafio_bancario_basico.py`: código do primeiro desafio, com funções para depósitos, saques e extrato.

## Nota

Esta versão foca na lógica básica do sistema bancário e serve como base para a evolução do próximo desafio.
